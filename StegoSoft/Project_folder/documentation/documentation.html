1. Introduction

Steganography:

Steganography is the art and science of writing hidden messages in such a way that no one, apart from the sender and intended recipient, suspects the existence of the message. Steganography, in general, relies on the imperfection of the human auditory and visual systems.

StegoSoft is a software which implements steganography using audio files. This software can hide message files inside a audio file by using the Least Significant Bit Algorithm.

2. Summary of Features

The features provided by StegoSoft is described as follows:

  1. Multiple messages file support in a single audio file.

  2. The message can be of any type i.e., an image, text, video, etc.

  3. Password protection and Encryption.

  4. Optimized Least Significant Algorithm. It ensures low distortion in quality of audio file after hiding the message file.

  5. User-friendly Graphical User Interface.

3. Operations

Hide:

The Hide operation is the process of embedding message files into the master audio file. This process will create a Carrier file. The carrier file contains both the audio and the message files. However, the size of carrier file remains the same as master file.

Unhide:

The Unhide operation is the reverse of Hide operation. This process extracts the embedded message files from the carrier file.

4. Terminology

Master File:

Master file is the source audio file in which you intend to hide any message. It can be a file in wav audio format only (This application supports only uncompressed audio formats).

Message File:

Message files are the files which you want to hide inside a master file. StegoSoft supports hiding of any file-type inside a master file. You can also store multiple message files inside a master file. The max limit for the number of files in a master file is 127. The total size of message files should be approximately 1/10th the size of master file. If this criteria is not fulfilled, the hide operation will fail.

Password:

The password is a added security. You can add a password while performing a hide operation. The same will be required for the unhide operation. When a password is selected, the message file will be stored in encrypted form.

Carrier File:

Carrier file is the audio file generated after a hide operation. This file contains both the audio and the message files.

5. Using StegoSoft

5.1 The Hide Form




  1. Click on 'Browse' to select the master file.

  2. Click on 'Add File' to add message files one by one. The same will be shown in the table once selected.

  3. If there is any file selected by mistake, select the same on the table and click 'Remove File'.

  4. Check the 'Use Password' check-box if you want to add a password.

  5. If you selected 'Use Password', enter password for the same.

  6. Click on 'HIDE'. It will ask you the location and file-name for the carrier file.

Back to Index



5.2 The Unhide Form




  1. Click on 'Browse' to select the carrier file.

  2. Enter the password in the text box inside the magnifier.

  3. Click on 'Show Files' to see the file-name of message files stored inside the carrier file.

  4. Click on 'UNHIDE' to extract the message files stored inside the carrier file. It will prompt you for a folder location where the files will be extracted.
